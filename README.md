# 我的响应式网站

一个使用 HTML5 和 CSS3 构建的响应式网站，支持电脑和手机自适应显示。

## 特性

- 📱 响应式设计，适配各种屏幕尺寸
- 🎨 瀑布式布局 (Masonry Grid)
- 🌊 流式响应布局
- 🎯 左侧栏导航
- 📱 移动端优化的汉堡菜单

## 使用方法

直接在浏览器中打开 `index.html` 即可查看效果。

## 部署到 GitHub Pages

1. 将项目推送到 GitHub 仓库
2. 进入仓库设置 -> Pages
3. Source 选择 "main branch"
4. 保存后即可通过 `https://用户名.github.io/仓库名` 访问

## 目录结构

```
my-website/
├── index.html    # 主页面
├── style.css     # 样式文件
├── script.js     # JavaScript
└── README.md     # 说明文件
```
